Project description

The application consist of a menu and three pages:

  • Home - project description,

  • Login - signin/signup forms,

  • About - profile page example.

It is build on Angular 4 with Angular cli using.

Features description

Signin/signup/send a message

  • On 'login' page you are able to sign in or sign up. I set up a firebase project, so signin/signup/signout are real actions and all users and user data will be saved on the firebase database.

  • Also you will be auto-logged in if your token is still stored in localstorage.

  • Depends on a user status some elements will be shown or hidden:

    • Login/logout form,

    • Contact form on 'About' page.

  • On 'About' page if you are logged in you are able to send a message which also will be stored in the firebase DB.

Form validation

All forms validated via angular Validator and show errors if something goes wrong.

Multi-languages support

  • You can switch languages and last language will be saved with cookies using.

  • I created a custom '| trans' pite which works with json and trying to find matches depends on current language. Also it supports internal variable objects.

  • To show translations like initial variable add '?display_keys' parameter to the url (e.g. https://mono-app-aa73a.firebaseapp.com/about?display_keys).

Services

  • Flash banners service: common banner service to show status messages e.g. when you create an account or send a message;

  • Global variables servise: root variables for the whole project.

Dynamic cards

On 'About' page after XHR we get a json with reviews and build review cards.
